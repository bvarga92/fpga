- **audio_fir**: kétcsatornás FIR szűrő szoftverből programozható együtthatókkal ([videó](https://www.youtube.com/watch?v=uIjuZ1wRhi4&list=PL9_VlVdB8s882QMHiqJlDpJeKWxwP5CIG))
